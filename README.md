# Redux Shop

 <h2>
        A redux Shop app with FakeStore API. Fetched the data from FakeStore
        with axios. Used Redux for the app functions. You can choose an Shop
        Item and go to it's details by router.
      </h2>
      <h3>
        I Used:
        <ul>
          <li>HTML</li>
          <li>CSS</li>
          <li>Semantic UI</li>
          <li>JS</li>
          <li>React</li>
          <li>React Hooks</li>
          <li>Redux</li>
          <li>Axios</li>
          <li>Router</li>
        </ul>
      </h3>
